## ✨ 核心特性

### 📚 **阅读标记系统** ⭐ 全新功能

4种智能标记类型，让阅读更系统化：

- 🟡 **Highlight** - 普通高亮标记
- ⭐ **Important** - 重点内容标记
- ❓ **Question** - 疑问内容标记
- 📝 **Note** - 笔记内容标记

**功能亮点：**
- ✅ 快捷键支持（Ctrl+H / Ctrl+Shift+H）
- ✅ 标记列表管理和查看
- ✅ 按类型筛选标记
- ✅ 导出为Markdown格式
- ✅ 云端同步，跨设备访问

### ⚡ **极速翻译**

- **并行请求技术** - 同时请求多个API，取最快结果
- **智能缓存** - 重复翻译瞬间完成（<1ms）
- **速度提升 2-5倍** - 平均翻译时间从1.8秒降至0.6秒
- **超时保护** - 5秒超时，避免无限等待

### 🎓 **专业术语保护**

- **150+ AI/ML术语词典** - 深度学习、机器学习、NLP等
- **智能识别** - 自动识别并保护专业术语
- **双向支持** - 中译英、英译中都准确
- **Tech/AI模式** - 一键切换专业翻译

### 📌 **固定窗口模式**

- **Always on Top** - 窗口始终置顶
- **专为PDF设计** - 阅读PDF的最佳伴侣
- **不自动关闭** - 可连续翻译
- **拖动定位** - 自由调整窗口位置

### 🔊 **双向朗读**

- **原文朗读** - 支持中英文
- **译文朗读** - 支持中英文
- **自动语言检测** - 智能选择发音
- **视觉反馈** - 朗读时动画提示

### ✨ **豹纹美学**

- **野性优雅** - 独特的豹纹主题设计
- **流畅动画** - 爪印动画、尾巴摆动等

## 🚀 快速开始

### 1. 网页翻译

```
选中文字 → 自动显示翻译框
或
按 Ctrl+Q 翻译选中文字
```

### 2. 阅读标记

```
1. 点击扩展图标
2. 选择标记类型（🟡/⭐/❓/📝）
3. 启用「Enable Marker Mode」
4. 选中文字后按 Ctrl+H
5. 点击「View All」查看标记
6. 点击「Export」导出为MD文件
```

### 3. PDF翻译

```
1. 点击扩展图标
2. 点击「📌 Pin Window」
3. 窗口保持置顶
4. 从PDF复制文字（Ctrl+C）
5. 粘贴到翻译窗口（Ctrl+V）
6. 一键翻译！
```

### 4. 专业翻译

```
1. 点击扩展图标
2. 选择「🎓 Tech/AI」模式
3. 输入/粘贴技术文本
4. 享受准确的专业翻译！
```

## 💡 常见问题

### Q: 如何翻译PDF文件？

A: 使用「固定窗口模式」：
1. 点击扩展图标
2. 点击「📌 Pin Window」
3. 复制PDF文字粘贴到窗口
4. 窗口始终置顶，方便使用

### Q: 如何导出标记？

A: 
1. 点击「📋 View All」
2. 点击「💾 Export as Markdown」
3. 自动下载MD文件

### Q: 翻译不准确？

A: 
- 普通文本：使用 **General** 模式
- 技术文档：使用 **🎓 Tech/AI** 模式

### Q: 标记会丢失吗？

A: 不会！标记保存在Chrome同步存储中，登录Chrome账号后跨设备同步。
