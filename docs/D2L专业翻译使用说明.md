# D2L专业翻译功能使用说明

## 🎯 功能概述

小豹翻译现已针对《动手学深度学习》(Dive into Deep Learning, D2L) 进行了专业化优化，提供更准确的学术和技术翻译。

## 🚀 新增功能

### 1. 专业翻译模式

在翻译界面中，您现在可以选择以下翻译模式：

- **General**: 通用翻译模式
- **🎓 Tech/AI**: 技术/AI领域专业翻译
- **📚 Academic**: 学术论文翻译模式
- **🧠 D2L**: 专门针对D2L教材的翻译模式

### 2. 智能网站检测

当您在以下网站使用划词翻译时，系统会自动启用D2L专业模式：

- `d2l.ai`
- `zh.d2l.ai` 
- `en.d2l.ai`
- 任何标题包含"Dive into Deep Learning"或"动手学深度学习"的页面

### 3. 扩展专业术语词典

新增了大量D2L相关的专业术语，包括：

#### 线性代数

#### 优化算法
- convex optimization → 凸优化
- saddle point → 鞍点

#### 网络架构

## 📖 使用方法

### 方法1：手动选择模式
1. 打开小豹翻译插件
2. 在翻译模式中选择"🧠 D2L"
3. 输入要翻译的文本
4. 点击翻译按钮

### 方法2：自动检测（推荐）
1. 在D2L相关网站上直接选中文本
2. 系统会自动使用D2L专业模式进行翻译
3. 翻译结果会保持专业术语的准确性

### 方法3：快捷键翻译
- 选中文本后按 `Ctrl+Q` 进行翻译
- 在D2L网站上会自动使用专业模式

## 🎯 翻译效果对比

### 普通模式 vs D2L专业模式

**原文**: "The backpropagation algorithm computes gradients by applying the chain rule of differentiation."

**普通模式**: "反向传播算法通过应用微分的链式法则来计算梯度。"

**D2L专业模式**: "反向传播算法通过应用微分的链式法则来计算梯度。"
*（术语保护：backpropagation → 反向传播，gradients → 梯度，chain rule → 链式法则）*

**原文**: "We implement a multilayer perceptron with two hidden layers using automatic differentiation."

**普通模式**: "我们使用自动区分实现了一个具有两个隐藏层的多层感知器。"

**D2L专业模式**: "我们使用自动微分实现了一个具有两个隐藏层的多层感知机。"
*（术语保护：multilayer perceptron → 多层感知机，automatic differentiation → 自动微分）*

## 🔧 技术特性

1. **术语保护机制**: 专业术语在翻译过程中被保护，确保准确性
2. **上下文感知**: 根据翻译模式添加专业化提示，提高翻译质量
3. **智能缓存**: 相同内容的翻译结果会被缓存，提高响应速度
4. **多API并行**: 同时调用多个翻译API，选择最快的结果

## 💡 使用建议

1. **学习D2L教材时**: 建议使用D2L专业模式，确保术语的一致性
2. **阅读学术论文时**: 使用Academic模式，保持学术表达的准确性
3. **技术文档翻译**: 使用Tech/AI模式，适合一般技术内容
4. **日常翻译**: 使用General模式即可
