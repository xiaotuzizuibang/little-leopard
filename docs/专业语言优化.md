# Cross-Lingual Technical Text Generation without Parallel Corpora

## 1. Background
Traditional machine translation depends heavily on parallel corpora. In specialized domains such as computer science, parallel corpora are limited, terminology inconsistencies are common, and consumer translation tools often degrade technical terms into everyday words.

Transformer-based models can learn deep semantic representations solely from monolingual data. This raises a central question: can a model trained only on English technical content and structured knowledge produce coherent Chinese technical text without relying on standard bilingual alignment?

## 2. Objective
Investigate whether a model can generate understandable and semantically accurate Chinese technical language while being trained primarily on:

- English technical corpora  
- Domain knowledge graphs and structured definitions  
- Minimal generic Chinese text (non-technical)

The generated terminology does not need to follow existing Chinese conventions. It only needs to remain internally consistent and semantically faithful.

## 3. Research Questions
1. Can consistent Chinese technical terminology emerge **without parallel corpora**?  
2. How stable will the emergent terminology be across contexts?  
3. Does structured knowledge improve cross-lingual semantic mapping?  
4. How does the emergent text compare with MT systems and human translators?

## 4. Methodology

### 4.1 Data
- English-only technical corpora (OS, networking, ML, distributed systems…)  
- Domain-specific knowledge graphs  
- Minimal Chinese corpus for general language modeling

### 4.2 Model
Evaluate two configurations:

- Baseline multilingual Transformer, fine-tuned only on English domain text  
- Knowledge-grounded model, integrating graph embeddings or schema supervision

### 4.3 Evaluation
- **Terminology emergence:** clustering, coherence metrics  
- **Semantic correctness:** expert review, concept alignment  
- **Readability:** comprehension tests with Chinese technical readers  
- **Comparison:** standard MT systems + human translators  

### 4.4 Experiments
- Zero-shot Chinese generation  
- Technical summarization  
- Cross-lingual Q&A  
- Terminology stability under paraphrasing

## 5. Expected Contributions
- Demonstrate coherent technical Chinese generation without parallel data  
- Show how structured knowledge improves alignment  
- Provide a new framework for terminology emergence in low-resource MT  
- Enable more robust domain-specific translation applications

## 6. Impact
This approach may reduce reliance on expensive parallel corpora, improve terminology stability, and support automatic technical documentation in emerging or specialized fields.
