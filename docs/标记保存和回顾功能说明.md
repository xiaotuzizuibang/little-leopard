# 📚 标记保存和回顾功能说明

## 🎯 功能概述

你的标记会自动保存到 Chrome 浏览器的同步存储中，支持跨设备同步（如果你登录了 Chrome 账号）。这样你可以随时回顾之前标记的内容。

---

## 💾 标记保存机制

### **自动保存**
- ✅ 每次标记都会**自动保存**到 Chrome Storage
- ✅ 支持 **Chrome 账号同步**（跨设备访问）
- ✅ 保存的信息包括：
  - 标记的文本内容
  - 标记类型（💡学到了 / ❓不理解 / ✂️跳过）
  - 标记时间
  - 来源网页URL
  - 可选的笔记

### **存储位置**
- 使用 `chrome.storage.sync` API
- 容量限制：100KB（大约可存储 500-1000 条标记）
- 自动与 Chrome 账号同步

## 📖 查看标记的三种方式

### **方式1：弹出窗口快速查看**

1. 点击浏览器工具栏的小豹图标
2. 点击 **"📋 View All"** 按钮
3. 在弹出的面板中查看所有标记

**功能：**
- 🔍 搜索标记内容
- 📑 按类型筛选（💡/❓/✂️）
- 📑 按 URL 分组查看
- 📋 复制标记内容
- 🔗 跳转到原网页
- 🗑️ 删除单个标记

---

### **方式2：独立历史页面（推荐）**

**打开方式：**
- 在弹出窗口点击 **"📖 History"** 按钮
- 或者直接在浏览器地址栏输入：`chrome-extension://[扩展ID]/history.html`

**特色功能：**

#### 📊 **统计看板**
```
┌─────────────────────────────────────┐
│  Total: 128  |  💡: 65  |  ❓: 38  |  ✂️: 25  │
└─────────────────────────────────────┘
```
一目了然地查看你的学习统计

#### 🔍 **强大的搜索和筛选**
- 全文搜索：搜索标记内容、笔记、URL
- 类型筛选：只看学到的、只看疑问等
- URL分组：按网站/文章分组显示

#### 📱 **卡片式展示**
- 每个标记以精美卡片展示
- 支持网格布局，一屏看多个
- 悬停效果，操作直观

#### 💾 **数据管理**
- 导出 Markdown：生成结构化笔记
- 导出 JSON：完整备份数据
- 导入 JSON：恢复备份或合并标记
- 一键清除：清理所有标记

---

### **方式3：独立固定窗口**

1. 点击浏览器工具栏的小豹图标
2. 点击 **"📌 Pin Window"** 按钮
3. 在新窗口中查看，支持置顶显示

**适用场景：**
- PDF 阅读时同时查看标记
- 双屏操作时将标记窗口放在副屏
- 需要长时间参考标记内容

- ✅ 在家、在公司都能查看
- ✅ 换电脑也不丢失

## 📊 数据统计分析

### **学习效率分析**

通过统计看板可以：
- 查看总标记数：反映阅读量
- 查看 💡 占比：反映收获
- 查看 ❓ 占比：发现薄弱点
- 查看 ✂️ 占比：了解过滤能力

### **健康比例参考**

```
理想状态：
💡 学到了: 40-50%  ← 高质量内容
❓ 不理解: 15-25%  ← 适度挑战
✂️ 跳过:   25-35%  ← 有效筛选

如果：
💡 < 20%  → 内容太简单或不认真
❓ > 40%  → 内容太难，需补基础
✂️ > 50%  → 内容质量不高
```
