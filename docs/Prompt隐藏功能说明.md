# 🧹 Prompt 隐藏功能说明

## 🎯 功能概述

在翻译时，系统会根据选择的模式自动添加上下文提示（Prompt）来提高翻译质量，但这些提示**不会**显示在最终的翻译结果中。

---

## 📋 工作原理

### **翻译流程**

```
用户输入文本
    ↓
添加上下文提示（仅发送给翻译 API）
    ↓
获取翻译结果
    ↓
✨ 自动清除提示前缀 ✨
    ↓
显示干净的翻译结果
```

### **示例**

**翻译模式：** 🧠 D2L（动手学深度学习）

**内部处理：**
```
发送给 API：
[《动手学深度学习》专业翻译：保持数学公式、算法名称、技术术语的准确性，使用教材标准译名] 深度学习是机器学习的一个分支。

API 返回：
[《动手学深度学习》专业翻译：保持数学公式、算法名称、技术术语的准确性，使用教材标准译名] Deep learning is a branch of machine learning.

自动清理后：
Deep learning is a branch of machine learning.
```

**你看到的：**
```
✅ 只显示干净的翻译结果
❌ 不显示任何提示信息
```

## 🔍 清理机制

### **智能识别模式**

系统使用多个正则表达式模式来识别和清除提示：

```javascript
✅ 识别中文提示：
- [技术文档翻译]
- [学术论文翻译，...]
- [《动手学深度学习》...]
- [AI/机器学习专业翻译，...]
- [高密度技术文档翻译，...]

✅ 识别英文提示：
- [Technical Translation]
- [Academic Paper Translation...]
- ["Dive into Deep Learning"...]
- [AI/ML Professional Translation...]
- [High-density Technical Translation...]

✅ 通用模式（兜底）：
- [任何包含中英文、数字、标点的方括号内容]
```

### **多次迭代清理**

```
第1次: 移除第一个匹配的提示
第2次: 检查是否还有提示
第3次: 继续清理...
...
最多5次: 防止无限循环
```

### **示例：多重提示**

```
输入: [提示1] [提示2] 实际内容
第1次清理: [提示2] 实际内容
第2次清理: 实际内容
最终输出: 实际内容 ✅
```

## 🔧 技术细节

### **清理函数位置**

```
文件: background.js
函数: removeContextPrefix(text)
调用位置:
  - translateWithGoogleAPI()
  - translateWithMyMemory()
  - translateWithLibreTranslate()
```

### **清理时机**

```
翻译 API 返回结果后
    ↓
第1次清理: 移除上下文前缀
    ↓
清理翻译文本: 移除多余空白
    ↓
恢复术语: 还原被保护的专业术语
    ↓
第2次清理: 再次移除可能产生的格式问题
    ↓
返回给用户
```

### **性能考虑**

```
✅ 正则表达式高效匹配
✅ 最多5次迭代（防止死循环）
✅ 只在需要时清理
✅ 不影响翻译速度
```

## 🐛 故障排查

### **问题 1：提示仍然显示**

**症状：** 翻译结果中看到 [《动手学深度学习》...] 等提示

**解决方法：**

1. **刷新扩展**
   ```
   chrome://extensions/ → 点击刷新 ⟳
   ```

2. **检查清理函数**
   ```javascript
   // 在控制台测试
   const testText = '[《动手学深度学习》专业翻译] 测试';
   // 应该看到清理逻辑
   ```

3. **查看控制台日志**
   ```
   F12 → Console → 查找 "🐆" 开头的日志
   ```

### **问题 2：过度清理**

**症状：** 正常的方括号内容也被清除

**示例：**
```
输入: 深度学习 [1] 是机器学习的分支
错误输出: 深度学习  是机器学习的分支 ❌
正确输出: 深度学习 [1] 是机器学习的分支 ✅
```

**解决方法：**
- 当前的清理模式只匹配开头的提示
- 不会影响文本中间的方括号
- 如果出现问题，请报告具体案例

### **问题 3：清理不完整**

**症状：** 部分提示残留

**示例：**
```
输入: [提示A] [提示B] 内容
错误输出: [提示B] 内容 ❌
```

**解决方法：**
- 已实现多次迭代清理
- 最多清理5次
- 如果仍有残留，可能是特殊格式
