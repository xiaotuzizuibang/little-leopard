## ✨ 学习标记快捷键（全新升级）

### 一键标记，自动启用

| 快捷键 | 功能 | 标记类型 | 颜色 |

## 🔧 快捷键冲突处理

### Ctrl+D 可能和浏览器书签冲突

**解决方案：**
1. 扩展快捷键优先级更高，会先触发
2. 如果需要添加书签，先取消选中文字
3. 或者使用 `Ctrl+Shift+D` 添加书签

### Ctrl+Q 可能和系统快捷键冲突

**解决方案：**
- Windows：通常无冲突
- Mac：如果冲突，使用 `Cmd+Q` 代替
- Linux：如果冲突，可能需要修改系统快捷键

## ❓ 常见问题

### Q: 快捷键不生效？

**A: 检查以下几点：**
1. ✅ 是否在支持的网页上（PDF可能不支持）
2. ✅ 是否重新加载了扩展（修改后需要刷新）
3. ✅ 是否选中了文字（Ctrl+L/D 需要选中文字）
4. ✅ 刷新页面试试

### Q: 使用快捷键后没有提示？

**A: 检查以下：**
1. 标记会立即显示（高亮或删除线）
2. 首次使用会有"标记模式已启用"提示
3. 如果看不到，可能是页面样式冲突

### Q: 可以自定义快捷键吗？

**A: 暂不支持自定义**
- Ctrl+L/Q/D 是精心设计的
- L = Learned，Q = Question，D = Delete
- 左手就能操作，右手用鼠标选择
- 黄金组合！💎

### Q: 可以同时使用多种标记吗？

**A: 可以！**
- 不同段落可以用不同标记
- 同一段落只能有一种标记
- 重新标记会覆盖旧标记

### Q: 标记会保存吗？

**A: 自动保存！**
- 实时同步到 Chrome storage
- 刷新页面后依然存在
- 按网址分类存储
- 可以导出为 Markdown
