# ✂️ 删除标记功能说明

## 🎯 功能概述

全新的**删除标记功能**，让您可以直接划掉不需要的文字片段，标记为"已读/不重要/待删除"内容！

---

## ✨ 功能特点

### 视觉效果

- ✂️ **删除线样式** - 红色删除线，清晰可见
- 🌫️ **半透明效果** - 降低不重要内容的视觉干扰
- 🎨 **淡灰背景** - 区别于其他标记类型

### 使用场景

#### **场景1：阅读长文时**
```
遇到不重要的段落 → 选中 → Ctrl+D → 划掉
视觉上淡化，保持阅读流畅性
```

#### **场景2：筛选信息时**
```
第一遍：标记所有重要内容 ⭐
第二遍：划掉不需要的内容 ✂️
最后：只关注未标记的内容
```

#### **场景3：学习笔记时**
```
已掌握的知识点 → 划掉 ✂️
还需复习的内容 → 保持高亮 🟡
重点难点 → 标记重要 ⭐
```

#### **场景4：文档编辑建议**
```
建议删除的段落 → 用删除标记 ✂️
导出Markdown → 自动带删除线格式
便于他人查看编辑建议
```

---

## 🚀 使用方法

### 方式1：快捷键（推荐）

```
1. 启用标记模式
2. 选中要划掉的文字
3. 按 Ctrl+D
4. 文字立即显示删除线效果
```

### 方式2：通过面板

```
1. 点击扩展图标
2. 选择 "✂️ Delete" 标记类型
3. 在网页上选中文字
4. 按 Ctrl+H 添加删除标记
```

## 📥 Markdown导出

### 导出格式

删除标记在导出的Markdown中自动转换为删除线语法：

```markdown
## ✂️ Delete (5)

### 1. ~~This is an outdated concept~~

**Source:** [example.com](https://example.com)
*Marked on: 2024/12/20 10:30:00*

---

### 2. ~~This paragraph can be removed~~

> 📝 **Note:** Not relevant to current research

---
```

## ⌨️ 快捷键总结

| 快捷键 | 功能 | 适用场景 |
|--------|------|----------|
| `Ctrl+H` | 🟡 高亮 | 一般重要内容 |
| `Ctrl+Shift+H` | ⭐ 重点 | 核心关键内容 |
| `Ctrl+D` | ✂️ 删除 | 不需要的内容 |
| `Alt+Click` | 移除标记 | 误操作后恢复 |

## 💡 使用技巧

### 技巧1：三色标记法

```
🟡 黄色高亮 - 可能有用的内容
⭐ 红色重点 - 一定要记住的内容
✂️ 灰色删除 - 已经不需要的内容
```

**适合**：
- 信息筛选
- 重点提取
- 进度管理

### 技巧2：GTD式管理

```
初读：
- 全部不标记（Inbox）

第一遍处理：
- 🟡 值得关注（Next Action）
- ✂️ 不需要关注（Trash）

第二遍处理：
- ⭐ 核心重点（Important）
- 📝 需要笔记（Reference）
- ❓ 存在疑问（Waiting）
```

### 技巧3：阅读流程优化

```
Step 1: 快速浏览
- 用 ✂️ 划掉明显无关的内容
- 视觉上淡化，减少干扰

Step 2: 精读
- 聚焦未标记的内容
- 添加 🟡/⭐/❓ 标记

Step 3: 复习
- 隐藏 ✂️ 标记的内容
- 只看重点和疑问
```

## ⚠️ 注意事项

### 注意1：不是真删除

删除标记**不会真正删除网页内容**，只是视觉标记：
- ✅ 内容仍然存在
- ✅ 可以随时恢复（Alt+Click）
- ✅ 不影响原网页

### 注意2：导出时的处理

在Markdown中：
- ✂️ 标记的内容会带删除线 `~~text~~`
- 但不会真的被删除
- 方便审查和对比

### 注意3：与翻译的配合

- 可以先翻译再标记
- 也可以先标记再翻译
- 删除标记不影响翻译功能
