# 📄 PDF 翻译完整解决方案

## ⚠️ 为什么本地PDF仍然无法划词翻译？

即使您已经：
- ✅ 启用了"允许访问文件网址"
- ✅ 重新加载了扩展
- ✅ 刷新了PDF页面

**仍然无法使用的原因：**

Chrome的PDF查看器是一个**特殊的内置组件**（Chrome PDF Viewer），它使用独立的渲染进程，与普通网页完全不同。Chrome出于安全考虑，**严格限制了扩展对PDF查看器的访问**。

### 🔍 技术原因

```
普通网页：浏览器 → 网页DOM → 扩展可以注入
PDF文件：浏览器 → PDF查看器（隔离环境）→ 扩展无法访问PDF内容层
```

即使是官方扩展如Google Translate，也无法直接在Chrome PDF查看器中工作！

## ✅ 实际可行的解决方案

### 🌟 **方案A：使用扩展弹窗（最推荐）**

**操作步骤：**
1. 在PDF中选中文字，按 `Ctrl+C` 复制
2. 点击浏览器工具栏的 🐆 小豹翻译图标
3. 在输入框中粘贴（`Ctrl+V`）
4. 点击"Translate"按钮

**优点：**
- ✨ 100%可用，无任何限制
- ✨ 支持最多1000字
- ✨ 美观的豹纹界面
- ✨ 支持朗读功能
- ✨ 一次可以翻译整段/整页内容

**快捷操作：**
```
Ctrl+C（PDF中复制） 
→ 点击扩展图标 
→ Ctrl+V（粘贴） 
→ Ctrl+Enter（翻译）
```

## 🛠️ 重新安装扩展（彻底排查）

如果您想确保扩展配置正确：

### 完整重装步骤：

1. **卸载扩展**
   - 打开 `chrome://extensions/`
   - 找到"小豹翻译"
   - 点击"移除"

2. **重新加载**
   - 点击"加载已解压的扩展程序"
   - 选择扩展文件夹

3. **检查权限**
   - 点击扩展的"详细信息"
   - 确认以下权限已启用：
     - ✅ "读取和更改您在所有网站上的数据"
     - ✅ "允许访问文件网址"（如需翻译本地PDF）

4. **测试**
   - 在普通网页上测试划词翻译 ✅
   - 在PDF中测试扩展弹窗翻译 ✅
