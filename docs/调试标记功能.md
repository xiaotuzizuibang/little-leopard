# 🐛 调试标记功能

## 问题诊断步骤

### 步骤 1：测试标记创建

1. 打开测试页面：`test-markers.html`
2. 选中一段文字
3. 按 `Ctrl+L` 添加标记
4. 查看是否有高亮显示
5. 打开浏览器控制台（F12），查看日志

**预期日志：**
```
📚 Marker saved to global list: {id: "...", text: "...", ...}
```

### 步骤 2：检查 Storage

在控制台中运行：
```javascript
chrome.storage.sync.get(['markers'], (result) => {
  console.log('Markers in storage:', result.markers);
  console.log('Count:', result.markers?.length || 0);
});
```

### 步骤 3：测试 View All

1. 点击浏览器工具栏的 🐆 图标
2. 查看标记计数是否正确：`📋 View All (数字)`
3. 点击 "View All" 按钮
4. 查看控制台日志

**预期日志：**
```
📚 Marker system initialized
📚 Loaded X markers
```

### 步骤 4：查看历史页面

1. 点击 "📖 History" 按钮
2. 应该打开新标签页显示所有标记
3. 检查统计数字是否正确

## 常见问题排查

### 问题 1：标记没有保存

**症状：** 标记后刷新页面，标记消失

**检查：**
1. 打开控制台，查看是否有错误
2. 运行：
```javascript
chrome.storage.sync.get(null, (items) => {
  console.log('All storage:', items);
});
```

**可能原因：**
- Storage API 权限问题
- 浏览器同步未开启
- Storage 容量已满

### 问题 2：View All 显示为空

**症状：** 点击 View All 看不到标记

**检查：**
1. 确认 markers 已保存到 storage
2. 确认 markerManager 已初始化
3. 在 popup 控制台运行：
```javascript
markerManager.markers
```

**修复方法：**
1. 刷新扩展：chrome://extensions/ → 点击刷新按钮
2. 重新打开 popup
3. 清除 storage 重新开始：
```javascript
chrome.storage.sync.clear(() => {
  console.log('Storage cleared');
});
```

### 问题 3：标记计数不准确

**症状：** 显示的数字与实际不符

**检查：**
```javascript
// 在 popup 控制台
markerManager.loadMarkers();
setTimeout(() => {
  console.log('Markers:', markerManager.markers.length);
}, 500);
```

**修复方法：**
- 调用 `markerManager.updateMarkerCount()` 强制更新

### 问题 4：搜索功能不工作

**症状：** 搜索框输入没有反应

**检查：**
1. 打开 View All 面板
2. 在搜索框输入
3. 查看控制台是否有错误
4. 检查事件监听是否绑定：
```javascript
document.getElementById('markerSearchInput')
```

## 强制刷新步骤

如果功能不正常，尝试：

1. **刷新扩展**
   - 打开 `chrome://extensions/`
   - 找到 "Little Leopard Translator"
   - 点击刷新图标 ⟳

2. **清除并重新加载**
   ```javascript
   // 在任意页面控制台
   chrome.storage.sync.clear(() => {
     console.log('Cleared!');
     location.reload();
   });
   ```

3. **重新安装扩展**
   - 删除扩展
   - 重新加载解压的扩展

## 数据检查工具

在控制台运行以下命令检查数据：

```javascript
// 查看所有 markers
chrome.storage.sync.get(['markers'], (result) => {
  console.table(result.markers);
});

// 查看所有 highlights
chrome.storage.sync.get(['highlights'], (result) => {
  console.log('Highlights by URL:', result.highlights);
});

// 查看 storage 使用情况
chrome.storage.sync.getBytesInUse(null, (bytes) => {
  console.log(`Storage used: ${bytes} / 102400 bytes (${(bytes/1024).toFixed(2)} KB)`);
});

// 统计标记类型
chrome.storage.sync.get(['markers'], (result) => {
  const markers = result.markers || [];
  const stats = {
    total: markers.length,
    learned: markers.filter(m => m.type === 'learned').length,
    question: markers.filter(m => m.type === 'question').length,
    strikethrough: markers.filter(m => m.type === 'strikethrough').length
  };
  console.log('Marker Statistics:', stats);
});
```

## 完整重置

如果一切都不工作，完全重置：

```javascript
// 清除所有数据
chrome.storage.sync.clear(() => {
  chrome.storage.local.clear(() => {
    console.log('All storage cleared!');
    // 然后刷新扩展并重新加载页面
  });
});
```
