# 🐆 小豹翻译 - 完整使用指南

## 📦 安装步骤

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 打开右上角的「开发者模式」开关
4. 点击「加载已解压的扩展程序」
5. 选择本扩展的文件夹（包含 manifest.json 的文件夹）
6. 安装完成！🎉

## 🎯 功能概览

### 1. 网页划词翻译
- 在任何网页上选中文字，自动弹出翻译框
- 翻译框固定显示，不会自动关闭
- 可以连续翻译多个词句
- 拖动标题栏可以移动翻译框位置
- 点击 × 按钮或按 ESC 键关闭

### 2. 智能学习标记
**无需手动启用！选中文字后直接使用快捷键：**
- `Ctrl+L` → 💡 学到了（标记新知识）
- `Ctrl+Q` → ❓ 不理解（标记疑问）
- `Ctrl+D` → ✂️ 删除（划掉不需要的）
- `Alt+点击` → 移除单个标记

### 3. 专业翻译模式
- **General**: 普通翻译模式
- **🎓 Tech/AI**: 技术/AI专业翻译（150+术语保护）
- **📚 Academic**: 学术论文翻译模式
- **🧠 D2L**: 专门针对《动手学深度学习》的翻译模式

### 4. 📌 固定窗口模式（推荐用于PDF阅读）
- 点击扩展图标 → 点击「📌 Pin Window」按钮
- 翻译器在新窗口中打开，始终保持在最上层
- 完美配合PDF阅读，窗口不会被遮挡

## 📄 PDF 文件翻译解决方案

⚠️ Chrome的PDF查看器有限制，扩展无法直接在PDF中选词翻译

### 【方法1：固定窗口模式】（最佳体验 ⭐⭐⭐⭐⭐）
1. 点击扩展图标 → 点击「📌 Pin Window」按钮
2. 翻译器在新窗口打开，固定在屏幕右侧
3. 窗口始终保持在最上层，不会被其他窗口遮挡
4. PDF中复制文字（Ctrl+C）
5. 翻译窗口自动显示在最前面，粘贴翻译（Ctrl+V）
6. 无需切换窗口，直接可见

### 【方法2：普通弹窗翻译】（快速简单）
1. 在PDF中选中并复制文字（Ctrl+C）
2. 点击扩展图标打开翻译面板
3. 粘贴到输入框，点击翻译

## ⌨️ 快捷键列表

### 学习标记（直接使用，无需预先设置）
- `Ctrl+L` - 💡 标记为 Learned（学到了）
- `Ctrl+Q` - ❓ 标记为 Question（不理解）*
- `Ctrl+D` - ✂️ 标记为 Delete（删除）
- `Alt+点击` - 移除单个标记

### 翻译功能
- `Ctrl+Q` - 打开翻译框（没有选中文字时）*
- `Ctrl+Enter` - 在弹窗输入框中翻译
- `ESC` - 关闭翻译框

*注：Ctrl+Q 智能判断：有选中文字→标记；无选中文字→打开翻译

## ❓ 常见问题

**Q: 为什么PDF无法划词翻译？**
A: Chrome的PDF查看器是独立环境，扩展无法注入。请使用固定窗口模式（方法1）

**Q: 标记功能不工作？**
A: 直接使用快捷键（Ctrl+L/Q/D）即可，会自动启用标记模式

**Q: 翻译不准确？**
A: 普通文本使用General模式；技术文档请使用Tech/AI模式；D2L内容会自动使用专业模式

**Q: 如何导出标记？**
A: 点击扩展图标 → 点击"📋 View All" → 点击"💾 Export as Markdown"

**Q: 看到 ⚡ 图标是什么意思？**
A: 这表示使用了缓存，翻译几乎瞬间完成！
