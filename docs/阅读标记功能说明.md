# 📚 小豹翻译 - 阅读标记系统

## 🎯 功能概述

全新的**阅读标记系统**，让您在阅读时轻松标记重点、疑问、笔记等内容，完美适合学习和研究！

---

## ✨ 核心功能

### 1. **四种标记类型**

| 类型 | 图标 | 用途 | 快捷键 |
|------|------|------|--------|
| **Highlight** | 🟡 | 普通高亮，标记一般内容 | `Ctrl+H` |
| **Important** | ⭐ | 重点标记，标记关键内容 | `Ctrl+Shift+H` |
| **Question** | ❓ | 疑问标记，标记不理解的内容 | - |
| **Note** | 📝 | 笔记标记，可添加个人笔记 | - |

### 2. **标记管理**

- ✅ **查看所有标记** - 点击"View All"按钮查看
- ✅ **按类型筛选** - 按标记类型快速过滤
- ✅ **复制标记** - 一键复制到剪贴板
- ✅ **删除标记** - 单个或批量删除
- ✅ **自动保存** - 云端同步，永不丢失

### 3. **导出功能**

- 📥 **Markdown格式导出**
- 📊 **按类型分组**
- 🔗 **保留来源链接**
- 📅 **包含时间戳**

---

## 🚀 使用方法

### **方式1：通过面板标记**

```
1. 点击扩展图标打开面板
2. 选择标记类型（🟡/⭐/❓/📝）
3. 启用"Enable Marker Mode"
4. 在网页上选中文字
5. 按 Ctrl+H 添加标记
```

### **方式2：快捷键标记**

```
Ctrl+H        → 快速高亮（黄色）
Ctrl+Shift+H  → 重点标记（红色）
Alt+Click     → 删除标记
```

### **查看标记列表**

```
1. 点击扩展图标
2. 点击 "📋 View All (数量)" 按钮
3. 查看所有标记
4. 可以按类型筛选、复制、删除
```

### **导出标记**

```
1. 打开标记列表
2. 点击 "💾 Export as Markdown"
3. 自动下载MD文件
4. 可在任何Markdown编辑器中查看
```

## 🎯 最佳实践

### **标记原则**

1. **🟡 Highlight** - 用于一般重要的信息
2. **⭐ Important** - 用于最关键的内容（少用，保持重点）
3. **❓ Question** - 用于不理解的地方，便于后续复习
4. **📝 Note** - 用于需要添加个人理解的地方

### **管理建议**

1. **定期整理** - 每周导出一次，归档到笔记软件
2. **及时删除** - 已掌握的内容可以删除
3. **添加笔记** - 重要标记添加个人理解
4. **分类清晰** - 合理使用不同类型

### **导出建议**

1. **按主题导出** - 学习一个主题后导出一次
2. **重命名文件** - 导出后重命名为有意义的名称
3. **整合笔记** - 导入到Notion、Obsidian等笔记软件
4. **定期备份** - 防止数据丢失

## 💬 常见问题

### Q: 标记会丢失吗？

A: 不会！标记保存在Chrome同步存储中，登录Chrome账号后跨设备同步。
   建议定期导出备份。

### Q: 可以添加多少标记？

A: Chrome同步存储限制为100KB，约可存储2000-3000条标记。
   建议定期导出并清理旧标记。

### Q: 导出的Markdown文件如何使用？

A: 可以用任何Markdown编辑器打开，如：
   - Typora
   - Obsidian
   - Notion
   - VS Code
   - 甚至GitHub直接显示

### Q: 如何在PDF中使用标记？

A: 1. 使用固定窗口模式
   2. 从PDF复制文字
   3. 在翻译窗口中翻译
   4. 翻译后添加标记
   5. 在标记管理中查看

### Q: 可以修改标记的颜色吗？

A: 目前每种类型的颜色是固定的，保证视觉一致性。
   未来版本可能支持自定义配色。
