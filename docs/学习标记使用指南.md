# 📚 学习标记系统 - 使用指南

## 🎯 设计理念

**专为第一遍阅读设计**，帮助您高效标记学习内容：

- 💡 **学到了** - 标记学到的新知识
- ❓ **不理解** - 标记看不懂的地方
- ✂️ **跳过** - 标记不需要的内容

---

## 🚀 快速开始

### **三步完成第一遍阅读**

```
Step 1: 阅读时随手标记
  - 看到新知识点 → Ctrl+L 标记"学到了"
  - 遇到看不懂的 → Ctrl+Q 标记"不理解"
  - 不重要的内容 → Ctrl+D 标记"跳过"

Step 2: 完成后查看标记
  - 点击"View All"查看所有标记
  - 按类型筛选：💡/❓/✂️

Step 3: 导出整理
  - 点击"Export as Markdown"
  - 在笔记软件中复习
```

---

## 💡 标记类型详解

### 1. 💡 学到了 (Learned)

**适用场景：**
- 新概念、新知识点
- 重要的理论和方法
- 值得记住的例子
- 有启发的观点

**快捷键：** `Ctrl+L`

**视觉效果：**
- 黄色背景高亮
- 醒目但不刺眼

**使用建议：**
```
阅读技术文章：
- 新的API用法 → 💡
- 编程技巧 → 💡
- 最佳实践 → 💡

阅读论文：
- 关键发现 → 💡
- 核心公式 → 💡
- 创新点 → 💡
```

---

### 2. ❓ 不理解 (Confused)

**适用场景：**
- 看不懂的术语
- 不理解的概念
- 需要进一步学习的内容
- 待解决的疑问

**快捷键：** `Ctrl+Q`

**视觉效果：**
- 红色背景高亮
- 引人注意

**使用建议：**
```
第一遍阅读：
- 遇到不懂的直接标记 ❓
- 不要停下来查，保持阅读流畅性

第二遍复习：
- 专注查看 ❓ 标记
- 逐个解决疑问
- 理解后可以改为 💡 或删除标记
```

---

### 3. ✂️ 跳过 (Skip)

**适用场景：**
- 已经掌握的基础知识
- 不相关的内容
- 不重要的细节
- 可以跳过的例子

**快捷键：** `Ctrl+D`

**视觉效果：**
- 删除线 + 半透明
- 视觉淡化

**使用建议：**
```
阅读长文：
- 废话/铺垫 → ✂️
- 重复内容 → ✂️
- 离题内容 → ✂️

阅读教程：
- 已掌握的概念 → ✂️
- 不适用的例子 → ✂️
```

## 🎯 实际应用流程

### **场景1：阅读技术博客**

```
【第一遍：快速标记】
1. 开始阅读
2. 看到新知识 → Ctrl+L (💡)
3. 看不懂的 → Ctrl+Q (❓)
4. 废话水分 → Ctrl+D (✂️)
5. 15分钟完成全文

【第二遍：解决疑问】
1. 查看标记列表
2. 筛选 ❓ 标记
3. 逐个查资料、理解
4. 理解后改为 💡 或删除

【导出复习】
1. Export as Markdown
2. 导入到Notion/Obsidian
3. 只看 💡 标记复习
```

### **场景2：阅读深度学习论文**

```
【Abstract】
- 核心创新点 → 💡
- 不理解的概念 → ❓

【Introduction】
- 背景知识（已知）→ ✂️
- 问题定义 → 💡
- 解决方案概述 → 💡

【Method】
- 新方法详细描述 → 💡
- 复杂公式 → ❓
- 实现细节 → 💡

【Experiments】
- 关键数据 → 💡
- 消融实验细节 → ✂️

【总结】
- 最后导出整理，形成读书笔记
```

## 📊 标记统计分析

### **健康的标记比例**

```
💡 Learned: 30-40%
  - 说明文章信息密度高
  - 学到了很多新知识

❓ Confused: 10-20%
  - 有一些疑问很正常
  - 需要进一步学习

✂️ Skip: 30-50%
  - 过滤掉不重要内容
  - 提高阅读效率
```

### **标记比例诊断**

```
💡 < 10%
  ⚠️ 可能文章太简单或阅读不认真

❓ > 50%
  ⚠️ 可能文章太难，需要补充基础知识

✂️ > 70%
  ⚠️ 可能文章质量不高或不适合你
```

## 🔄 学习循环

### **三遍阅读法**

```
【第一遍：标记】
目标：快速浏览，标记重点和疑问
时间：15-20分钟
标记：💡 + ❓ + ✂️

【第二遍：理解】
目标：解决所有 ❓ 标记
时间：30-40分钟
方法：查资料、看例子、做笔记

【第三遍：复习】
目标：只看 💡 标记，快速回顾
时间：10分钟
效果：巩固记忆
```

## 📱 在不同场景下使用

### **PDF阅读 + 标记**

```
1. 使用固定窗口模式
2. 从PDF复制文字
3. 粘贴翻译
4. 翻译后直接标记：
   - 新知识 → 💡
   - 不懂 → ❓
5. 完成后导出笔记
```

### **网页文章 + 标记**

```
1. 启用标记模式
2. 边读边标记
3. 不影响翻译功能
4. 标记和翻译可以同时进行
```
