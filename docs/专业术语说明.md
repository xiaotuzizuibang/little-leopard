## 🎯 使用示例

### 示例1：翻译技术论文

**原文：**
```
The transformer architecture with self-attention mechanism 
has achieved state-of-the-art results in NLP tasks.
```

**使用 Tech/AI 模式翻译：**
```
Transformer架构配合自注意力机制在NLP任务中取得了最先进的结果。
```

✅ **正确保护了**：transformer, self-attention, NLP

---

### 示例2：翻译训练日志

**原文：**
```
Training the model with batch size 32 and learning rate 0.001,
using Adam optimizer and cross entropy loss function.
```

**使用 Tech/AI 模式翻译：**
```
使用批量大小32和学习率0.001训练模型，
使用Adam优化器和交叉熵损失函数。
```

✅ **正确保护了**：batch size, learning rate, Adam optimizer, cross entropy, loss function

---

### 示例3：中译英

**原文：**
```
我们使用卷积神经网络进行图像分割，
并通过数据增强来防止过拟合。
```

**使用 Tech/AI 模式翻译：**
```
We use convolutional neural networks for image segmentation,
and prevent overfitting through data augmentation.
```

✅ **正确保护了**：卷积神经网络, 图像分割, 数据增强, 过拟合

## 💡 最佳实践

### ✅ 推荐做法
1. **翻译技术文档时启用 Tech/AI 模式**
2. **连续翻译时保持模式不变**
3. **术语会自动识别和保护**

### ⚠️ 注意事项
1. 术语需要完整匹配（如 "neural network" 而非 "neural"）
2. 大小写不敏感（"BERT" 和 "bert" 都会识别）
3. 支持复合词（"convolutional neural network"）

## 🔧 如需添加新术语

如果您发现某些专业术语翻译不准确，可以：

1. 打开 `terminology.js`
2. 在 `TERMINOLOGY` 对象中添加：
```javascript
'your term': '您的术语',
```
3. 重新加载扩展
