# 🐆 小豹翻译 - 图标设计需求

## 设计概念

**主题：** 小豹子在看书/学习

**用途：** Chrome扩展程序图标

---

## 设计要求

### 核心元素

1. **小豹子（主体）**
   - 卡通风格，可爱友好
   - 特征明显：豹纹斑点
   - 表情：专注/开心看书
   - 姿势：拿着书本或坐着阅读

2. **书本（辅助元素）**
   - 打开的书
   - 或者小豹子抱着书
   - 简化设计，不要太复杂

3. **整体风格**
   - 扁平化设计（Flat Design）
   - 极简主义
   - 圆润可爱
   - 容易识别

---

## 配色方案

### 主色调（来自UI设计）

```
主色：金黄色 (#D4A017) - 豹子身体
辅色：深棕色 (#5B3A29) - 豹纹斑点、轮廓
点缀：浅黄色 (#FFF8E1) - 背景或书本
强调：橙黄色 (#FFE54C) - 高光或书本
```

### 色彩运用

- 背景：浅色或透明
- 主体：金黄色系
- 细节：深棕色
- 避免：过于复杂的渐变

---

## 尺寸规格

### Chrome扩展标准尺寸

| 尺寸 | 用途 | 细节程度 |
|------|------|----------|
| 16x16 px | 浏览器工具栏 | 极简，只保留核心形状 |
| 32x32 px | 任务栏/书签 | 简化，主要元素可见 |
| 48x48 px | 扩展管理页面 | 基本细节完整 |
| 128x128 px | Chrome商店展示 | 完整细节，高质量 |

### 技术要求

- 格式：PNG（带透明背景）
- 颜色：RGB模式
- 分辨率：72 DPI以上
- 背景：透明或纯色

## 设计细节建议

### 16x16 版本（极简）
```
- 只显示豹子头部轮廓
- 2-3个豹纹斑点
- 简单的眼睛和鼻子
- 不显示书本（太小）
```

### 32x32 版本（简化）
```
- 豹子头部 + 上半身
- 4-6个豹纹斑点
- 可以加一个小书本图标
- 保持轮廓清晰
```

### 48x48 版本（标准）
```
- 完整的豹子形象
- 清晰的豹纹图案
- 明确的书本元素
- 可以加眼镜或学习元素
```

### 128x128 版本（精细）
```
- 高质量渲染
- 丰富的细节
- 可以加阴影和高光
- 书本上可以有文字或图案
- 表情更生动
```

## 在线工具推荐

### 免费AI生成
1. **Bing Image Creator** (bing.com/create)
   - 基于DALL-E
   - 完全免费
   - 质量很好

2. **Leonardo AI** (leonardo.ai)
   - 免费额度
   - 多种风格
   - 易于使用

3. **Canva AI** (canva.com)
   - 内置AI生成
   - 免费模板
   - 可直接编辑

### 图标编辑工具
1. **Figma** (figma.com)
   - 在线使用
   - 矢量编辑
   - 导出多尺寸

2. **Canva** (canva.com)
   - 简单易用
   - 大量模板
   - 支持PNG导出

3. **Photopea** (photopea.com)
   - 类似Photoshop
   - 在线免费
   - 功能强大

### 图标库网站
1. **Flaticon** (flaticon.com)
   - 海量图标
   - 可自定义颜色
   - 免费下载

2. **Icons8** (icons8.com)
   - 多种风格
   - 在线编辑
   - 支持AI生成

3. **The Noun Project** (thenounproject.com)
   - 简洁图标
   - 高质量
   - 可商用
